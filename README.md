# Task Manager App

A modern, responsive task management application built with React.

## Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Filter tasks (All/Active/Completed)
- Persistent storage
- Clean, modern UI

## Installation

npm install
npm run dev

## Technologies
- React
- Vite
- Tailwind CSS
- Lucide Icons
